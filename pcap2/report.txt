Scenario:
The attacker appears to have identified a web application running on our server. We need to determine what application was being targeted. Find the version and username associated with the application in the capture.


Writeup:
Used Wireshark export  object "http" to find wp "WordPress" traffic, then followed it to find Web server and version.

Used Wireshark filter "http contains "ver="", followed the response, and found version 1.0.9 and tested up to 6.8

Used Wireshark filter "http.request.method == "POST"": found Username and E-mail
